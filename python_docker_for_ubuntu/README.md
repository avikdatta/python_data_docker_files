# Python data analysis on Ubuntu

A Docker image for Python data analysis on Ubuntu 16.04. 
Installed Tools:

    Pyenv
    Python 3.5.2
    Pandas
    Scipy
    Numpy
    Scikit-learn
    Matplotlib
    Seaborn
    Jupyter-notebook
    Pymysql
    Beautiful soup 4
