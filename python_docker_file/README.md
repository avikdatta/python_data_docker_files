# Python data analysis on Raspberry pi

A Docker image for Python data analysis on Raspberry pi. This image is created using resin/rpi-raspbian.

## Installed Tools:
 * Pyenv
 * Python 3.5.2
 * Pandas
 * Scipy
 * Numpy
 * Scikit-learn
 * Matplotlib
 * Seaborn
 * Jupyter-notebook
 * Pymysql
